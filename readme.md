Welcome to **Learn Golang**! This repository contains hands-on code examples for learning the Go programming language (Golang). It is designed for beginners and intermediate learners who want to master Go's core concepts through practical, well-organized examples.

## 💡 About Go
Go (or Golang) is an open-source programming language designed for simplicity, reliability, and efficiency. It is widely used for backend development, cloud services, and more.

## ⚡ Getting Started

### Prerequisites
- 🛠️ Install [Go](https://golang.org/dl/) (version 1.18 or later recommended)

### Running Examples
To run any example, navigate to its directory and use the `go run` command:

```bash
cd basics
go run main.go
```

Replace `basics` and `main.go` with the folder and file you want to run.
